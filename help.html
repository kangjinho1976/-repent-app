🙏 기도하자! 완벽 사용 설명서

회개 기도 노트 V1.3.9 마스터 가이드
1. 왕초보를 위한 3단계 퀵 스타트

처음 앱을 켜셨다면, 아래의 3단계만 기억하시면 됩니다!


1 기도 대상 & 나이 설정
중앙 메뉴에서 [이름 설정]을 눌러 기도를 드릴 대상(예: 홍길동, 외가 등)을 추가하고 선택합니다. 그 후 [나이 설정]을 눌러 기도를 시작할 나이나 대(代)를 맞춥니다.

2 기도문 불러오기
[기도문 목록] 버튼을 누르고 미리 저장된 기도문을 선택합니다. (직접 하단의 입력창에 쓰셔도 됩니다.)

3 기도 시작!
화면 맨 밑의 초록색 [다음 단계 ▶] 버튼을 누르면, 타이머가 돌아가며 기도가 시작됩니다! 한 살(또는 한 대)씩 올라갈 때마다 버튼을 다시 눌러주세요.

2. ✨ 마법의 자동 변환 기호 사용법

기도문 안에 특수한 기호를 섞어 쓰면, 매번 글자를 고칠 필요 없이 앱이 알아서 이름과 나이를 바꿔가며 읽어줍니다. (괄호 모양을 정확히 쓰셔야 합니다!)

👤 이름 자동 변환 : <이름>

기도문에 <홍길동> 이라고 적어두면, [이름 설정]에서 선택한 사람의 이름으로 싹 바뀝니다. 특히, 뒤에 조사(이/가)를 붙여두면 받침 유무에 따라 알아서 예쁘게 바뀝니다.

[작성 예시] 주님! <아무개>이 지은 죄를...

✅ 대상을 '이순신'으로 선택 시 ➔ 주님! 이순신이 지은 죄를...

✅ 대상을 '김철수'로 선택 시 ➔ 주님! 김철수가 지은 죄를...

🎯 나이 자동 증감 : [ ]살

가운데에 빈칸을 두고 [ ]살 이라고 쓰면, 앱이 알아서 '한살, 두살' 같은 우리말 나이로 예쁘게 바꿔서 읽어줍니다.

[작성 예시] [ ]살 때 우상숭배한 죄

✅ 1살 진행 시 ➔ 한살 때 우상숭배한 죄

✅ 20살 진행 시 ➔ 스무살 때 우상숭배한 죄

🌳 가문/조상 대(代) 자동 증감 : [ ]대

조상님을 위한 기도를 할 때 [ ]대 라고 쓰면, '1대, 2대' 처럼 숫자로 자연스럽게 바꿔줍니다.

[작성 예시] 친가 [ ]대 조상님이 지은 죄

✅ 3대 진행 시 ➔ 친가 3대 조상님이 지은 죄

3. 📊 심층 통계 및 음성(TTS) 활용법

[통계 자료] 메뉴 활용하기

  • 나의 목표: 하루에 몇 분 기도할지 목표를 정할 수 있습니다. (기본 30분) 진행률 바를 보며 동기부여를 받으세요!
  • 심층 분석: 어떤 사람(대상)을 위해, 어떤 기도문으로, 오늘/이번 달/올해 각각 몇 번을 기도했는지 3단계로 정밀하게 보여줍니다.
  • 연속 달성일: 매일 꾸준히 기도하면 '연속 기도 중!' 일수가 올라갑니다.

[🗣️ TTS 음성 읽기] 활용하기

기도가 시작된 후, 입력창 상단의 [🔇 TTS 음성읽기 : OFF] 버튼을 누르면 기계가 기도문을 자동으로 또박또박 읽어줍니다. 글씨를 보기 힘들 때 눈을 감고 소리에 맞춰 기도해보세요!

  • [🗣 TTS 음성설정] 메뉴에서 읽는 속도(배속)와 목소리의 종류(성별 등)를 취향에 맞게 바꿀 수 있습니다.
4. 💾 [필독] 데이터 영구 보존 꿀팁
🔒 오프라인 철통 보안 앱입니다!

이 앱은 인터넷 서버(클라우드)로 회원가입을 하거나 데이터를 빼가지 않습니다. 선생님이 작성하신 모든 기도문과 통계는 오직 '내 스마트폰 기기 안'에만 저장됩니다.


⚠️ 스마트폰을 바꾸거나, 앱 데이터를 초기화하면 모든 기도문이 날아갑니다!

따라서, 소중한 기도문들을 안전하게 지키려면 꼭 아래 방법을 사용하세요.


주기적인 백업 방법 (파일로 저장하기)

  1. 중앙 메뉴에서 [기도문 관리]를 누릅니다.
  2. [📤 파일로 저장하기]를 누릅니다.
  3. '나의_회개_기도문.txt' 라는 파일이 스마트폰의 '다운로드' 폴더에 저장됩니다. 이 파일을 카카오톡 내게쓰기나 이메일로 보내두시면 평생 안전합니다!
  4. 새 스마트폰을 샀을 때, 이 텍스트 파일을 다운받은 뒤 [📥 파일에서 가져오기]를 누르면 모든 기도문이 1초 만에 완벽하게 복원됩니다!
5. 📱 스마트폰 홈 화면에 앱 설치하기

이 앱(웹앱)을 스마트폰 바탕화면에 설치하여 진짜 앱처럼 편리하게 쓰는 방법입니다.

  • 삼성폰(갤럭시) / 크롬 브라우저:
    화면 우측 상단의 [점 3개 메뉴(⋮)]를 누른 뒤, [홈 화면에 추가] 또는 [앱 설치]를 누르세요.
  • 아이폰(애플) / 사파리 브라우저:
    화면 하단의 [공유하기(네모와 위쪽 화살표 아이콘)]를 누른 뒤, 위로 스크롤하여 [홈 화면에 추가]를 누르세요.

이제 스마트폰 바탕화면에 생긴 '기도하자!' 아이콘을 누르면, 인터넷 주소창 없이 전체 화면으로 쾌적하게 기도를 드릴 수 있습니다.
